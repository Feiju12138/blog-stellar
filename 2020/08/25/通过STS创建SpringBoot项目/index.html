【笔记】通过STS创建SpringBoot项目
2020-08-25

前言

基于SpringToolSuite4创建SpringBoot项目

准备工作

  • Java 1.8
  • Maven 3.6.3

STS中Maven的配置

  • Spring Tool Suite 4->Preferences->Maven->Installations->Add

  • 配置Maven安装目录

  • 选中新的配置->应用

  • Maven->User Settings->修改配置->应用

通过官方的Starter创建项目

  • File->New->Spring Starter Project

  • 配置项目参数->Next

  • 选择SpringBoot版本->Finish

通过阿里云的Starter创建项目

  • File->New->Spring Starter Project

  • 配置项目参数，将Server URL改为阿里云的Starter->Next

  • 如果版本显示错误，可以手动调整版本，在配置版本处不管版本号，选择Next

  • 手动调整版本号->Finish

通过官网打包导入项目

  • 在官网配置项目参数，下载打包的新项目https://start.spring.io

SNAPSHOT：快照版本
M2：增加重要功能的版本，不是正式版

  • 解压，导入项目到STS

  • File->Inport->Maven->Existing Maven Projects->Next

  • 选择项目根目录->Finish

通过普通Maven项目创建项目

  • 创建一个简单的Maven项目

  • File->New->Other->Maven->Maven Project->选择简单Maven项目->Next

  • 配置项目参数 ->Finish

  • 替换pom.xml文件内容
  • 右键项目->Maven->Update Project

  • 选中Force Update of Snapshots/Releases强制更新->OK

  • 手动创建入口程序

  • 新建Java文件，全局限定名为com.company.CgbSpringboot04Application

  • 在类上定义注解@SpringBootApplication，并定义main方法

通过在其他机器的新项目复制

  • 通过以上任何方式构建完整新项目后，压缩项目拷贝到其他机器

  • 在pom.xml文件内调整参数

完成
