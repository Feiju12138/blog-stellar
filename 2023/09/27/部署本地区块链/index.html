【笔记】部署本地区块链
2023-09-27

前言

Quickly fire up a personal Ethereum blockchain which you can use to run tests, execute commands, and inspect state while controlling how the chain operates.（官网）

部署本地区块链（私有链），模拟以太坊区块链环境，用于开发和测试DApp

安装Ganache

通过Npm

直接下载GUI可执行程序

  • trufflesuite/ganache-ui

启动Ganache

连接测试网络

  • 打开MetaMask->网络->添加网络

  • 手动添加网络

  • 配置本地网络->保存

  • 切换至本地测试网络

添加测试账号

  • 切换账户->导入账户

  • 粘贴测试账号的私钥->导入

  • 导入成功默认有1000ETH的余额
