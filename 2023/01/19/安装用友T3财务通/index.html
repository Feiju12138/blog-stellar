【笔记】安装用友T3财务通
2023-01-19

前言

安装用友T3财务通普及版

准备工作

  • 在自动安装SQL Server数据库时，要求必须有.NET 2.0环境
  • 在安装用友T3时，要求系统主机名必须不含中文和特殊字符，且不能以数字开头
    • 如果是系统默认主机名，只需要将-去掉即可

安装用友T3

安装SQLServer数据库

  • 如果是已经部署好SQL Server环境，可以跳过此步骤；如果没有部署好SQL Server环境，可以通过用友T3自助安装程序进行安装

  • 启动Autorun.exe自助安装程序

    • 如果是Windows 10就选择SQL2005EXPRESS安装SQL Server 2005
    • 如果是Windows 7或以下版本就选择MSDE2000安装SQL Server 2000

此时如果没有安装.NET 2.0环境，可能会报错

  • 等待自动安装完成即可

安装主程序

  • 启动Autorun.exe自助安装程序，选择

  • 下一步

  • 勾选我接受许可协议中的条款->下一步

  • 设置用户名和公司名称，可以使用默认值->下一步

  • 设置安装路径，可以修改盘符，但是软件根目录必须为UFSMART->下一步

  • 选择需要安装的组件->下一步

  • 此时自动跳出环境监测，如果不符合环境要求需要部署环境和修改配置（如果是单机使用，可以不部署IIS环境，直接退出检测即可）

  • 下一步

  • 等待安装完成

  • 安装完成后选择完成，此时会自动重启，此时无需任何操作，等待重启完成

  • 重启后无需任何操作，等待自动弹出系统管理，完成安装

  • 完成安装后关闭提示窗口，即可开始使用

首次使用需要注册，注册时可以使用管理员用户直接登录

用户名：admin
密码：空

踩坑

  • 报错：未处理的异常情况 错误号码：0x800A0146 描述：Resource with identifier '138' not fround 安装程序将立即终止

原因

  • SQL Server版本过高，换低一点的版本

解决问题

  • 用友T3自助安装程序进行SQL Server的安装

完成
