【踩坑】Java项目运行报错

2023-03-17

前言

Java项目运行报错：java: 无效的目标发行版: 17

原因

  • 项目指定的Java版本有误

解决问题

  • 重新指定Java版本为8

  • SpringBoot不要超过3版本，3版本不支持Java8，并在项目中指定Java版本为1.8
